# Test Git Bundle Workflow

This repo simulates the git bundle workflow for BullzAI releases.

## Structure

- `mock-repos/` - Simulated bullzai-app and bullzai-data repos
- `releases/` - Generated release packages
- `tohad-workspace/` - Simulates Tohad's workspace
- `ops/` - Implementation of bundle creation logic
- `tasks.py` - Invoke tasks
